## 1. 项目概述

**my-wallet-butler** 是一款个人财务管理系统，帮助用户全面记录和管理个人资产、消费支出、收入来源，提供清晰的财务状况视图和智能分析能力。

### 1.1 核心目标

- 📊 **全面记账**：记录每一笔收入与支出
- 💰 **资产管理**：管理多类型资产（现金、银行卡、信用卡、数字货币、贵金属、不动产、车产等）
- 🌍 **多币种支持**：支持人民币、美元、欧元、日元等多种法币及贵金属
- 📈 **财务分析**：可视化报表，消费趋势分析
- 🎯 **预算管理**：设置月度/分类预算，超支提醒
- 🔒 **数据安全**：本地优先，可选云端同步

## 2. 功能需求

### 2.1 交易记录模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 新增交易 | 支持收入/支出，关联账户和分类 | P0 |
| 编辑/删除 | 修改历史记录，支持批量操作 | P0 |
| 交易详情 | 金额、时间、分类、账户、备注、标签、附件（截图/小票） | P0 |
| 批量导入 | 支持 CSV/Excel 导入，银行流水解析 | P1 |
| 重复交易 | 设置周期性自动记录（如月租、工资） | P1 |
| 多币种交易 | 支持不同币种的交易记录，自动汇率转换 | P0 |

### 2.2 账户管理模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 账户类型 | 现金、储蓄卡、信用卡、数字货币、投资账户、贵金属、不动产、车产等 | P0 |
| 多币种支持 | 支持人民币、美元、欧元、日元等多种法币 | P0 |
| 贵金属账户 | 黄金、白银、铂金等贵金属持仓管理 | P1 |
| 不动产账户 | 房产、土地等不动产估值管理 | P1 |
| 车产账户 | 车辆等动产估值管理 | P1 |
| 账户余额 | 实时更新，支持手动调整 | P0 |
| 多账户转账 | 账户间转账记录（支持币种转换） | P0 |
| 账户分组 | 按类型/用途分组管理 | P1 |
| 账户冻结/归档 | 不活跃账户隐藏 | P2 |

### 2.3 分类管理模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 预设分类 | 餐饮、交通、购物、娱乐、住房、医疗、教育等 | P0 |
| 自定义分类 | 用户自由创建/修改分类 | P0 |
| 多级分类 | 支持一级/二级分类（如 餐饮 → 外卖） | P1 |
| 收入/支出分类 | 区分收入来源和支出类别 | P0 |

### 2.4 预算管理模块

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 月度预算 | 按分类设置月度支出上限 | P0 |
| 年度预算 | 年度总支出/收入目标 | P1 |
| 预算提醒 | 达到 80%、100% 时提醒 | P1 |
| 预算执行率 | 实时显示预算使用情况 | P1 |
| 预算调整 | 支持临时调整预算额度 | P2 |

### 2.5 报表分析模块

| 功能 | 说明 | 优先级 |
|------|------|--------|

| 趋势折线图 | 收支趋势变化 | P0 |
| 账户资产变化 | 各账户余额变化曲线 | P1 |

## 3. 非功能需求

### 3.1 性能要求
- 列表加载 < 500ms
- 报表生成 < 2s
- 支持 10 万+ 交易记录

### 3.2 安全要求
- 用户数据隔离
- 敏感数据加密存储
- HTTPS 传输
- 操作日志审计

### 3.3 可用性要求
- 响应式设计（桌面 + 平板 + 手机）
- 离线支持（PWA）
- 数据备份与恢复
